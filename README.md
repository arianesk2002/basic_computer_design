# 💻 Basic Computer Design

A digital logic and computer architecture project focused on designing and implementing a **Basic Computer System** from fundamental hardware components.

This project demonstrates how core computer architecture modules interact to execute instructions, transfer data, and perform arithmetic and logical operations. ⚙️

---

## 📌 Overview

The objective of this project is to design a functional **Basic Computer** by combining essential digital hardware components into a complete processing system.

The design explores low-level computer organization concepts such as:

- 🧠 CPU architecture
- 🗄️ Memory organization
- 🔄 Data transfer mechanisms
- ➕ Arithmetic and logic operations
- 🎛️ Control unit implementation
- 📡 Common bus architecture

## 🏗️ System Components

The computer system is built using several fundamental modules:

### 🧠 Control Unit
Responsible for generating control signals and coordinating system operations.

### ➕ Arithmetic Logic Unit (ALU)
Performs arithmetic and logical operations such as:

- Addition
- Subtraction
- AND
- OR
- NOT

### 🗃️ Registers
Used for temporary data storage and instruction handling.

Typical registers may include:

- Program Counter (PC)
- Instruction Register (IR)
- Accumulator (AC)
- Data Register (DR)
- Address Register (AR)

### 💾 Memory Unit
Stores instructions and data required for execution.

### 📡 Common Bus
Provides communication between registers, memory, and processing units.

---

## ⚙️ Instruction Cycle

The system follows the standard instruction execution process:

```text
Fetch
   ↓
Decode
   ↓
Execute
   ↓
Store Result
```

Each instruction is processed through a sequence of micro-operations controlled by the Control Unit.
